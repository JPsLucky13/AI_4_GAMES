Self Playing Pac-Man

1. Open the unity project and go to Assets->Assignments->FinalProject->Scenes and Open scene PacManMenu.
2. To Navigate the scene press the up and down arrow
keys to select your mode of preference. 1 player is a single
player mode with infinite levels and 3 lives.
2 player mode is local with infinite levels and each player gets 3 lives.
AI player starts the game is a self playing pac-man that uses Qlearning. The lives in this mode have been set to 100.
3. Press Enter to start the selected mode.
4. For non AI modes, the arrow keys move pac man in the corresponding direction.
5. Have fun.
